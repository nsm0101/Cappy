# CloseDose App UI Kit

Mobile-first dose-lookup product. Kit includes the home screen, dose-result screen, kid switcher, medication card, and bottom tab bar — all light + dark mode aware.

## Files
- `index.html` — interactive demo (tap a med to navigate to the dose screen; tap moon/sun in the top bar to switch theme)
- `Icon.jsx` — Lucide-style stroke icons
- `Screens.jsx` — `HomeScreen`, `DoseScreen`, `TabBar`, `KidPill`, `MedCard`
- `ios-frame.jsx` — device bezel
- `styles.css` — kit-scoped utilities (buttons, inputs, cards)

## Notes & caveats
- Dose math is illustrative (acetaminophen ~12.5 mg/kg). Do not use as production logic.
- Built without codebase access — please attach for fidelity pass.
